Study flow

A clear way to prepare

Use the starter sample, learn the exam vocabulary, then move through the full tests and review the mistakes that matter.

{% if user %} Open dashboard {% else %} Try free demo {% endif %} Preview Exam Words

1. Start with the sample

Use Starter Test 0 to understand the format before going into the full practice library.

2. Check Exam Words

Review key Danish/English exam terms and the Russian helper translation before harder tests.

3. Complete tests 1-13

Work through the structured tests with road photos, explanations, saved questions, and progress.

4. Use mixed practice

The extra mixed test gives 25 questions from across the library, reshuffled each time.

5. Review saved questions

Bookmark unclear questions during tests and return to them from Saved when you revise.

6. Repeat weak areas

After each result, review wrong answers first, then repeat the test after a short break.

Good order for most learners

Starter sample -> Exam Words preview -> tests 1-13 -> saved questions -> mixed practice.
